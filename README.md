## 1. 개요

- **프로젝트명:** ICOM (Ice Cream Order Manager)
- **목적:** 분산된 매입 내역(직접 발주 및 사후 엑셀 등록)을 통합 관리하여 실시간 지출 현황과 수익성을 분석하는 관리 시스템.
- **주요 목표:**
    - 아이스크림 직접 발주 기능 및 발주서 이미지(Image) 생성/공유.
    - 과자, 음료 등 기타 품목의 명세서(Excel/CSV) 대량 업로드 처리
    - 품목별 단가/마진 자동 계산 및 시각화된 통계 대시보드 제공.

## 2. 시스템 아키텍처

- **Backend:** .NET 8 (C#) 기반 **Clean Architecture**
    - `Domain`: 엔티티 및 핵심 비즈니스 로직
    - `Application`: 서비스 인터페이스, DTO, 유스케이스
    - `Infrastructure`: DB Context(EF Core), 리포지토리 구현, 외부 연동
    - `Api`: RESTful API 컨트롤러
- **Frontend:** Vue.js 3 (Vite), Tailwind CSS, Axios
- **Database:** MSSQL (Entity Framework Core - Code First)

## 3. 주요 기능

### 3.1. 상품 및 단가 관리

- **카테고리 관리:** 아이스크림, 과자, 음료 등 분류별 데이터 필터링 조회.
- **용어 및 계산 로직:**
    - **Box Price (박스단가):** 박스 단위 가격.
    - **Package Size (입수):** 박스당 포함된 낱개 수량.
    - **Unit Price (낱개 매입가):** `Box Price` / `Package Size` (자동 계산)
    - **Retail Price (판매가):** 소비자에게 판매하는 최종 가격.
    - **Margin (마진율):** (`Retail Price` - `Unit Price`) / `Retail Price` (자동 계산)
- **데이터 무결성:** 모든 금액 데이터는 `decimal` 타입을 사용하여 계산 오차 방지.

### 3.2. 매입 및 발주 프로세스

- **아이스크림:**
    - 화면 내 제품 검색 및 발주 수량 직접 입력.
    - **이미지 내보내기:** 확정된 발주 리스트를 이미지 파일(PNG/JPG)로 내보내기 (도매처 전송용).
- **과자/음료 등:**
    - 주문 후 수령한 명세서(Excel/CSV) 업로드.
    - 기존 상품 마스터와 매칭하여 매입 내역 자동 생성.

### 3.3. 통계 및 대시보드

- **월간 지출 분석:** 월별 총 매입 지출액 합산.
- **카테고리 비중:** 카테고리별 지출 비중(%) 시각화.

## 4. 개발 로드맵

- **Phase 1:** 아이스크림 발주 리스트 및 이미지 내보내기 기능
- **Phase 2:** 과자/음료 엑셀 업로드 파이프라인 구축
- **Phase 3:** 통합 통계 대시보드 및 지출 리포트
- **Phase 4:** 사용자 로그인 및 권한 관리 시스템 도입
- **Phase 5:** 모바일 최적화 및 실시간 재고 알림 서비스
